Getting Started

It looks like the Flask web application may not be running. Here's how to start it:

  1. Open a terminal window
  2. Navigate to the ProfileScope project directory:
    cd /Volumes/BackUp/Python\ Projects/Profile_Scope_AI/ProfileScope
  3. Start the Flask application:
    python -m app.web.app
    or
    python run.py
  4. Once the server is running, refresh this page or click the button below
Open ProfileScope App

Project Structure

ProfileScope is structured as follows:

  • app/web/app.py - Main Flask application
  • app/web/templates/ - HTML templates
  • app/web/static/ - Static files (CSS, JavaScript)
  • app/core/ - Core analysis engine
